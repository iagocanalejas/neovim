# Action

```sh

<leader>xx          # [Trouble] diagnostics
<leader>xw          # [Trouble] workspace
<leader>xr          # [Trouble] References

<leader>e           # [Telescope] file explorer
<leader>ee          # Default explorer

<leader>uu          # [Telescope] undo

<leader>uC          # Colorscheme

```

# Find

```sh

<leader><space>     # Find files
<leader>ff          # Other find files

<leader>fr          # Find recent files
<leader>fR          # Other find recent files

```

# Search

```sh

<leader>/           # Search in files

<leader>sb          # Search current buffer
<leader>sd          # Search diagnostic
<leader>sg          # Search grep (like <leader>/)
<leader>sh          # Search highlights
<leader>sk          # Search keywords

# Less useful
<leader>sc          # Search command history
<leader>sH          # Search help

```

# Goto

```sh

<leader>gk          # Go to keywords
<leader>gK          # Go to other keywords

```

# Git

```sh

<leader>gc          # [Telescope] commits
<leader>gs          # [Telescope] status

<leader>gcc         # [Fugitive] open

### Inside [Fugitive]
<leader>p           # Push
<leader>P           # Pull
<leader>t           # push -u origin

```
