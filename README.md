# Shortcuts

## Global
```
Project View:               <leader>pv
Undo Tree:                  <leader>u
Format File:                <leader>f
Format Selection:           <leader>ri
Replace Current Word:       <leader>s

Half Page Jump:             <C-i> | <C-d>

Open Packer:                <leader>vpp
Make File Executable:       <leader>x
Source File:                <leader><leader>
```

## Utils
```
Toggle Comment:             gcc | gc

Paste in selection:         <leader>p  (doesn't lose the copied buffer)
Copy in system clipboard:   <leader>y

Remove \n:                  J

Move selected line:         J | K
```

## Find
```
Harpoon add:                <leader>a
Harpoon menu:               <C-e>
Harpoon move:               <C-a> | <C-s> | <C-d> | <C-f>

Find file:                  <leader>pf
Search in files:            <leader>ps

Find git file:              <C-p>
```

## LSP

```
Rename:                     <leader>vrn
```

```
Git status:                 <leader>gs
```
